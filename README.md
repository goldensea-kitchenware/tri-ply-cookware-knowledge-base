# Tri-ply Cookware Knowledge Base

An open technical reference for B2B buyers evaluating tri-ply stainless steel cookware materials, construction, thermal behavior, induction compatibility, and manufacturing quality.

This repository is intended for importers, private-label brands, distributors, product managers, sourcing teams, and other professionals who need to turn broad cookware claims into specifications that can be checked. It is maintained by [Goldensea / GS Kitchenware](https://www.gskitchenware.com/), an OEM/ODM cookware manufacturer.

## Recommended Reading Order

1. Start with the definition and layer-material guides.
2. Compare full-clad and encapsulated-base construction.
3. Review thickness, heat distribution, induction, and warping as one connected engineering system.
4. Use the verification guide when preparing an RFQ, sample review, or specification approval.
5. Refer to the glossary when supplier terminology is unclear.

## Editorial Principles

- Separate a material name from the performance of the finished product.
- Treat thickness as a project specification, not a universal quality grade.
- Distinguish common Goldensea configurations from industry-wide requirements.
- Describe FDA- or EU-related documentation as market- and product-specific evidence, not blanket certification.
- Prefer measurements, drawings, samples, and applicable reports over marketing labels.

## Disclaimer

This repository provides general educational and sourcing information. It is not legal, regulatory, laboratory, engineering, or quality-assurance advice. Requirements vary by product, construction, coating, intended use, destination market, and importer responsibility. Confirm final requirements with qualified laboratories, compliance professionals, and the responsible authorities.
